Framework Tailwind

Tailwind es un framework para CSS que te permite crear interfaces modernas y responsivas sin escribir CSS personalizado. En lugar de crear clases con estilos en un archivo externo, Tailwind te da clases predefinidas que aplicas directamente en tu codigo de HTML.

¿Qué es Material Design?

Material Design es un lenguaje de diseño desarrollado por la empresa de Google. Se basa en sombras reales, profundidad, jerarquía visual y transiciones suaves. El objetivo es crear interfaces consistentes y estéticamente agradables en múltiples plataformas. Material Design promueve el uso de una paleta de colores clara, componentes reutilizables, y animaciones que ayuden al usuario a entender mejor la interfaz.

Comandos básicos de Tailwind

Estas son algunas de las clases más comunes en Tailwind CSS que se utilizan para dar estilo directamente en el HTML:

  • bg-blue-500: cambia el color de fondo
  • text-white: cambia el color del texto
  • text-2xl: cambia el tamaño del texto
  • p-4: aplica padding (espaciado interno)
  • m-2: aplica margen (espaciado externo)
  • rounded: bordes redondeados
  • shadow: agrega una sombra simple
  • flex justify-center: centra contenido horizontalmente usando Flexbox
  • hover:bg-blue-700: cambia el fondo al pasar el mouse (hover)
  • transition: aplica animaciones suaves al cambiar de estado
